## Alternatif untuk menyimpan gambar yang lebih besar

Taruh gambar background login Anda di sini jika:
- Ukuran file > 500KB
- Tidak ingin di-bundle dengan Vite
- Ingin akses langsung via URL

### File yang direkomendasikan:
- `login-background.jpg`
- `login-background.png`

### Cara menggunakan:
Di `Login.jsx`, gunakan path:
```javascript
<img src="/images/login-background.jpg" alt="Login Background" />
```

**Note:** Path dimulai dari `/` (root public folder)
